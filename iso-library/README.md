# iso-library

Bu klasor, kullanicinin USB uzerinde tutmak istedigi Windows veya Linux ISO dosyalari icindir.

Beklenen kullanim:

- ISO dosyalarini bu klasore kopyala
- WinPE icinde CigerTool `Multiboot ISO` ekranindan otomatik goruntule

Not: Mevcut surum ISO kutuphanesini tarar ve listeler; USB boot menusu tarafinda daha ileri entegrasyon sonradan genisletilebilir.
